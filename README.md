# Figura Orb Model Template

<img width="427" height="452" alt="image" src="https://github.com/user-attachments/assets/ea3c8ee0-eeae-48e2-a51c-728d968ab1e2" />

Spherical player model template for [Figura 0.1.5](https://modrinth.com/mod/figura)

## ⚙️ Features
- **Rolling Movement**
- **Hitbox Scaling** (Requires Pehkui!)
- **Manual Rotation Reset**
- **Togglable First Person View**
- **Togglable Passive Gurgling**


## ❓ Necessary Dependencies for Hitbox Scaling

Requires Pehkui and/or Pehkui4All for hitbox scaling

**⚠ ENSURE Figura's "Chat Messages" setting is set to _ON_ for hitbox scaling to occur ⚠**
![Figura Chat Messages Setting](https://github.com/user-attachments/assets/210014bc-9efc-40c7-908d-6aba5c42277c)

## 🛠 Editing the Template Model

Figura only works with `.bbmodel` files made in Blockbench version [4.12.6](https://github.com/JannisX11/blockbench/releases/tag/v4.12.6) or earlier 

The transparent portions of the texture are to ensure the limbs can properly sink into the body, covering these will cause unintended covering of the limbs and head
